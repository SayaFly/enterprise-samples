Android Enterprise Samples Repository
=====================================

This repository contains a set of individual Android Studio projects to help you get
started writing/understanding Android Enterprise features.

## 📦 Download APKs

The latest APKs are built automatically on every push to the main branch:

- **ManagedConfigurations:** [Download](releases/apks/ManagedConfigurations.apk)
- **Work Profile Starter:** [Download](releases/apks/WorkProfile-Starter.apk)
- **Work Profile Finished:** [Download](releases/apks/WorkProfile-Finished.apk)

APKs are also available as GitHub Actions artifacts from the
[Build All APKs](../../actions/workflows/build-all-apks.yml) workflow.
